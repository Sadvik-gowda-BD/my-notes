In Java
ServerSocket - ServerSocket is a lower-level API in Java used to create a TCP server that listens for incoming connections.
HttpServer - HttpServer is a higher-level API provided by Java for creating HTTP servers. It simplifies the process of handling HTTP requests and responses.
